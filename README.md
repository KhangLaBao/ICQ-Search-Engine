
# 🌼 ICQ Search Engine

An unofficial recreation of the classic **ICQ Search** experience from the mid-2000s.

This project aims to recreate the nostalgic design of the original ICQ Search website while using modern search technology underneath.

---

## ✨ Features

- 🌼 Classic ICQ-inspired interface
- 🔎 Powered by Google Programmable Search Engine
- 📄 Lightweight HTML/CSS
- 💻 No frameworks
- ⚡ Fast loading
- 🌐 Designed for modern browsers
- 🎨 Faithful recreation of the original ICQ Search layout

---

## About

This project is **not affiliated with or endorsed by ICQ**.

It is a fan-made recreation created for nostalgia, preservation, and educational purposes.

The goal is to preserve the look and feel of one of the Internet's classic search portals while keeping it fully functional on today's web.

## Credits

This project uses Google Programmable Search Engine (Custom Search Engine) as its search backend.

The user interface is an original recreation inspired by the appearance of ICQ Search (2007), based on archived pages preserved by the Internet Archive's Wayback Machine.

Special thanks to the Internet Archive for preserving historical web pages that made recreating this interface possible.
