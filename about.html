🌟 About Red Star Search

Red Star Search is a retro-inspired search engine frontend built for the People. It is designed to be simple, lightweight, and accessible on all devices — from modern machines to legendary low-power systems.

🚩 Mission

In a world where modern websites become heavier and more complex, Red Star Search returns to a simpler era of the internet.

Fast loading. Clean interface. No unnecessary complexity.

⚙️ Technology

This project uses Google Programmable Search Engine for results, combined with custom styling and scripts to provide a classic experience.

Designed to work on:
• Old browsers (even legacy systems)
• Slow internet connections
• Mobile and WAP devices

📟 Features

• Retro UI inspired by early search engines
• Mobile (WAP) version for low bandwidth
• Custom “glorious results” display
• Editable Comrade username system
• Lightweight and fast

⚠️ Notes

Some modern features from Google may not function on very old browsers. However, the core search functionality remains fully operational.
